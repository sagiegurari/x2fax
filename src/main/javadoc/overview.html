Overview:

The fax4j x2fax is a set of standalone servers and deployable components which enable to provide x2fax services. Already comes with built-in email component to enhance email servers to provide email2fax services and HTTP servers and WARs to convert any web server to a web2fax bridge.
The x2fax is a sub project of the java fax library: fax4j

Installation/Setup:

Each server/component has it's installation/setup requirements.
Those installation instructions can be found in each server/component directory.
